# LocalMind - Vue.js Frontend

Vue.js 3 frontend using CDN (no build step required).

## Features
- Vue 3 Composition API
- Vue Router for navigation
- Reactive state management
- All CRUD operations
- Authentication with JWT

## Start

```bash
./start.sh
```

Then open: http://localhost:3000

## Structure
- index.html - Main HTML with Vue CDN
- app.js - Vue app with components and router
- style.css - Styles

## Usage
1. Register/Login
2. View questions
3. Create questions
4. Add comments
5. Toggle favorites
